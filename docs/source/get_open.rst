get_open\(\)
============

When called, this function will return a list of dictionaries containing
information about all of the currently open dining halls on campus. For
example:

.. code-block:: python

   [
       {
           "close": 1600,
           "open": 900,
           "name": "Patisserie"
       },
       {
           "close": 2000,
           "open": 800,
           "name": "Haines' Boulangerie"
       },
       {
           "close": 2200,
           "open": 700,
           "name": "M Emporium"
       },
       {
           "close": 2200,
           "open": 800,
           "name": "Miami Ice"
       },
       {
           "close": 2359,
           "open": 0,
           "name": "Pulley Diner"
       }
   ]
